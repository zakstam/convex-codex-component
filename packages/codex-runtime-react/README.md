# @zakstam/codex-runtime-react

React hooks and UI utilities for `@zakstam/codex-runtime`.

## Install

```bash
pnpm add @zakstam/codex-runtime-react
```

## Usage

```ts
import { CodexProvider, useCodex } from "@zakstam/codex-runtime-react";
```

`@zakstam/codex-runtime-react` depends on runtime/core contracts from `@zakstam/codex-runtime`.
