# @zakstam/codex-runtime-bridge-tauri

Tauri bridge package for `@zakstam/codex-runtime`.

## Install

```bash
pnpm add @zakstam/codex-runtime-bridge-tauri
```

## Usage

```ts
import { createTauriBridgeClient, generateTauriArtifacts } from "@zakstam/codex-runtime-bridge-tauri";
```

Node-only local adapter:

```ts
import { CodexLocalBridge } from "@zakstam/codex-runtime-bridge-tauri/local-adapter";
```
